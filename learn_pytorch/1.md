# 3. 张量的属性

## 3.1. shape

```python
torch.arange(4).shape
# torch.Size([4])


torch.arange(4).reshape(4, 1)
# torch.Size([4, 1])
```
## 3.2. 张量中元素的总数
```python
torch.arange(4).numel()
# 4
```
## 3.3. 张量中各元素相加
```python
torch.arange(4).sum()
# 6
```
# 4. 运算

## 4.1. 按元素（elementwise）

```python
x = torch.tensor([1.0, 2, 4, 8])
y = torch.tensor([2, 2, 2, 2])

# 向量相加，相减，按元素除法, 按元素乘法，按元素幂
x + y,      # tensor([ 3.,  4.,  6., 10.]),
x - y,      # tensor([-1.,  0.,  2.,  6.]),
x / y,      # tensor([0.5000, 1.0000, 2.0000, 4.0000]),
x * y,      # tensor([ 2.,  4.,  8., 16.]),
x ** y      # tensor([ 1.,  4., 16., 64.]))



x = torch.tensor([0, 1, 2, 3])
y = x*x
# (tensor([0, 1, 4, 9])

# e^x
torch.exp(x)
```
